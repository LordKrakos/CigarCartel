# Cigar Cartel

## Description
A store locator for Cigar Cartel locations

[Click Here](https://cigarcartel.net/)

## Built With
- HTML
- CSS, SCSS
- JavaScript, GSAP
- Python, Django,
- SQLite
- Google Maps API
- Webpack

## Features
- Store locator with Google Maps integration
- Responsive design for mobile and desktop
- Interactive map with store markers and info windows
- Search functionality to find stores by location
- Admin panel for managing store locations and details
